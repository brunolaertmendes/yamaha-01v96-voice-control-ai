# Yamaha 01V96 Voice Control AI

Projeto de automação da mesa Yamaha 01V96 usando:

- Python
- Inteligência Artificial
- Reconhecimento de voz
- MIDI
- Windows 10

## Objetivo

Controlar funções da mesa digital Yamaha 01V96 por comandos de voz.

## Tecnologias

- Python
- SpeechRecognition
- PyAudio
- mido
- python-rtmidi

## Exemplo de comandos

- "Aumentar canal 1"
- "Mutar canal 5"
- "Abrir retorno"
- "Baixar master"
